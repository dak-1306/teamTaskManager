# teamTaskManager

## Overview

Task Management System là ứng dụng quản lý công việc cho phép người dùng:

- Tạo và quản lý project
- Tạo và phân công task
- Theo dõi tiến độ công việc
- Quản lý thành viên trong project

## Tech Stack

Frontend

- React
- React Router
- Axios

Backend

- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Project Structure

```
project-root/
├── frontend/     # React application
├── backend/      # Node.js API server
└── README.md
```

## Installation

git clone <repo-url>
cd project-name

## Run Project

```bash
# chạy backend
cd backend
npm install
npm run dev

# chạy frontend
cd frontend
npm install
npm run dev
```
